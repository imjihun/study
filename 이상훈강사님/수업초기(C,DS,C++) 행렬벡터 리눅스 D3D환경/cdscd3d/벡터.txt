내적(스칼라값)(의미는 divergence 검색)
<v,w> = v*w*cos새타
두벡터가 서로 영향을 끼치는 양
	= v_x*w_x + v_y*w_y + v_z*w_z


외적(벡터값)
v X w = v*w*sin새타*(왼손법칙 or 오른손으로감는법칙)
x y = z
y x = -z
y z = x
z y = -x
z x = y
x z = -y

A x B = 
	i   j   k
	Ax  Ay  Az
	Bx  By  Bz 행렬의 diterminant를 구한다
-> [ (Ay*Bz-Az*By)i, (Az*Bx-Ax*Bz)j, (Ax*By-Ay*Bx)k ]
